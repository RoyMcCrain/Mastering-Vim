# バッファ

```bash
:ls
  2 %a   "memo.md"                      line 19
```

2 　バッファの ID
%はバッファが現在のウィンドウにある
a は active の意味。
ファイル名と現在のカーソル位置

```bash
:ls
  2 #a + "memo.md"                      line 31
  3 %a   "Chapter02/animals/cat.py"     line 6
```

バッファへのアクセスは`:b id`もしくは`:b ファイル名の一部`
ファイル名でいけるのは便利。候補が複数あるとエラーになる。
バッファの削除は`:bd`

# 折りたたみ　 folding

あんま使ったことないけどこれは Rails の annotate で追加されたコメントとか畳みたい。

| コマンド | 内容                 |
| -------- | -------------------- |
| `zo`     | 折りたたみを展開する |

| `za`     | トグル               |
| `zR`     | すべてを展開する     |
| `zM`     | すべてを折りたたむ   |

```vim
set foldmethod=syntax
set foldlevelstart=99
```

`autocmd BufRead * normal zR`はなんかうまくできなかったので上記で設定した。

# テキスト中を移動する

| コマンド | 内容                 |
| -------- | -------------------- |
| `{`      | 段落の先頭に移動する |
| `}`      | 段落の末尾に移動する |
| `(`      | 文章の先頭に移動する |
| `)`      | 文章の末尾に移動する |

移動のコマンドを図示

```
              gg
               ?  これは後方検索
            Ctrl-b
               H
               {
               k
^ F T ( b ge h   l w e ) t f $
               j
               }
               L
            Ctrl-f
               /  これは前方検索
               G
```

# インサートモード

# レジスタはどこ？

"とレジスタ識別子でレジスタにアクセスできる。
a から z までのレジスタは手動で設定する。
a レジスタにコピーするなら`ayw`
ペーストするには`ap`でする。
